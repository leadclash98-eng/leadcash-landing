## 4. Content Distribution Plan

### Primary Channels

| Channel | Strategy | Frequency |
|---------|----------|-----------|
| **Website Blog** | Publish all content, optimize for SEO | 2x/week |
| **LinkedIn (Company Page)** | Share blog posts with native preview + key takeaways | 2x/week |
| **LinkedIn (Yasiru's Personal)** | Thought leadership posts derived from blog content | 3-4x/week |
| **Email Newsletter** | Weekly digest of blog content to subscribers | 1x/week |

### Content Repurposing Strategy

Each blog post should be repurposed into:

1. **LinkedIn Carousel** - Key points from the post (designed by in-house team)
2. **LinkedIn Text Post** - Summarized takeaway or hot take from Yasiru
3. **Short-Form Video** - 60-90 second video summary (by Yohan)
4. **Email Snippet** - Featured in weekly newsletter
5. **Twitter/X Thread** - Key insights in thread format

### Promotion Checklist (Per Post)

- [ ] Publish on blog with proper SEO (meta title, description, OG image)
- [ ] Share on LinkedIn Company Page
- [ ] Yasiru shares personal take on LinkedIn
- [ ] Create carousel graphic for social media
- [ ] Include in weekly email newsletter
- [ ] Internal link from related existing blog posts
- [ ] Add to relevant pillar/category page

## 6. Technical SEO Requirements

### Blog Setup Checklist

- [ ] Create `/blog` route in Next.js app
- [ ] Set up MDX or CMS (Contentful, Sanity, or Notion as CMS)
- [ ] Implement blog listing page with pagination
- [ ] Create category/pillar pages for each content pillar
- [ ] Add breadcrumb navigation
- [ ] Implement schema markup (Article, BlogPosting, Organization)
- [ ] Add XML sitemap for blog posts
- [ ] Set up canonical URLs
- [ ] Implement OG meta tags and Twitter cards per post
- [ ] Add related posts section at bottom of each post
- [ ] Create author pages (Yasiru as primary author)
- [ ] Add table of contents for long-form content
- [ ] Implement lazy loading for images
- [ ] Add newsletter signup CTA in blog sidebar/footer
- [ ] Set up Calendly CTA widget on blog posts

### On-Page SEO Checklist (Per Post)

- [ ] Target keyword in title (H1)
- [ ] Target keyword in meta title (under 60 characters)
- [ ] Target keyword in meta description (under 160 characters)
- [ ] Target keyword in URL slug
- [ ] Target keyword in first 100 words
- [ ] Use H2 and H3 subheadings with related keywords
- [ ] Add internal links to relevant service pages and blog posts (3-5 per post)
- [ ] Add 1-2 external links to authoritative sources
- [ ] Optimize images with alt text containing keywords
- [ ] Include a clear CTA (Calendly booking or contact form)
- [ ] Add schema markup (BlogPosting)
- [ ] Ensure mobile responsiveness
- [ ] Target Core Web Vitals: LCP < 2.5s, INP < 200ms, CLS < 0.1

## Summary

This 6-month blog marketing strategy positions Lead Clash as a thought leader in B2B LinkedIn marketing, lead generation, and outreach services. By publishing 2 high-quality posts per week across 5 content pillars, the strategy targets:

- **48 total blog posts** covering all service areas
- **150+ keywords** ranking in the Top 50 by Month 6
- **15+ monthly leads** attributed to blog content
- **Industry-specific content** for all 5 target verticals
- **Full content repurposing** across LinkedIn, email, and video

The strategy prioritizes bottom-of-funnel, commercial-intent keywords alongside informational content to build topical authority and drive both organic traffic and qualified leads.
